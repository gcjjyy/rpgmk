# rpgmk
RPG쯔꾸르 DANTE PC의 16컬러 이미지 데이터를 읽어 출력하는 프로그램

# 빌드
```sh
$ brew install sdl2
$ make
```

# 사용법
```sh
$ ./imageview CHR00.DAT PALET.DAT 32 32
$ ./imageview ENEMY00.DAT PALET2.DAT 80 96
$ ./imageview OPNTITLE.DAT OPNTITLE.RGB 640 400
$ ./patnview
```
