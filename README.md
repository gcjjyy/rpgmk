# rpgmk
RPG쯔꾸르 DANTE PC의 16컬러 이미지 데이터를 읽어 출력하는 프로그램

# 빌드
```sh
$ brew install sdl2
$ make
```

# 사용법
```sh
$ ./rpgmk CHR00.DAT 8 32 32 PALET.DAT
$ ./rpgmk ENEMY00.DAT 1 80 96 PALET2.DAT
$ ./rpgmk OPNTITLE.DAT 1 640 400 OPNTITLE.RGB
```
